# docs/state-fields.md

> 本文档按系统维度整理 `js/state.js` 中的状态字段（含 `world`、`xikou`、顶层 `systems/research/techBonuses` 与历史数组）。每个字段一行说明用途。

## population（人口）

- `world.totalPopulation`：总人口，用于人口增长、需求和人均指标计算。
- `world.laborForce`：总劳动力规模。
- `world.children`：儿童人口规模。
- `world.elderly`：老年人口规模。
- `world.farmerPopulation`：农民阶层人口。
- `world.merchantPopulation`：商人阶层人口。
- `world.officialPopulation`：官员阶层人口。
- `world.workerPopulation`：工人阶层人口。
- `world.landlordPopulation`：地主阶层人口。
- `world.idleLabor`：当前未被分配的剩余劳动力。
- `world.studentsDownToVillage`：学子下乡人数设置。
- `xikou.population`：溪口总人口。
- `xikou.laborForce`：溪口劳动力。
- `xikou.children`：溪口儿童人口。
- `xikou.elderly`：溪口老年人口。
- `xikou.idleLabor`：溪口闲置劳动力。

## agriculture（农业）

- `world.farmlandAreaMu`：可耕地面积（亩）。
- `world.baseGrainYieldPerMu`：基础亩产。

- `world.grainDemandTotal`：总粮食需求。
- `world.grainAnnualDemand`：年度粮食总需求。

- `world.grainBalance`：年度粮食收支平衡值。
- `world.grainPerCapita`：人均粮食占有量。
- `world.grainCoverageRatio`：粮食覆盖率。
- `world.foodSecurityStatus`：粮食安全状态标签。
- `world.foodSecurityIndex`：粮食安全指数。
- `world.agriculturalTaxRate`：农业税率。
- `world.lastAgriculturalTax`：最近一次农业税额。
- `world.lastTaxCollectionYear`：最近税收结算年份。
- `world.grainTreasury`：可自由支配粮仓库存。
- `world.totalGrainDemand`：商业等系统带来的总粮食需求。
- `world.grainPrice`：粮价指数/价格。
- `world.supplyRatio`：粮食供给比。
